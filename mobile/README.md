# HomeCare365 Mobile

App Android & iOS (Capacitor) — mở website https://www.homecare365.vn trong WebView native.

## Nhanh

```bash
npm install
npm run check
npm run sync
npm run open:android   # hoặc open:ios (trên Mac)
```

Hướng dẫn đầy đủ: [docs/MOBILE_BUILD.md](../docs/MOBILE_BUILD.md)

**Không cần chỉ định đường dẫn lưu** — mặc định trong `mobile/` và `mobile/android/app/build/outputs/`.
